## AnalyseWriter, what is it?

analyseWriter is a free application released under the GNU GPL3 license. It is a console application. This application allows you to analyze and evaluate layout files in Open Document Format 1.2. (native format of LibreOffice and OpenOffice Writer).

analyseWriter allows:

     • analyze files created by (LibreOffice or OpenOffice) Writer and evaluate them.
     • compare the modifications in the files. This makes it possible to match in different files the same changes that took place at the same time.
     • generate reports for each file. The reports allow students to identify their mistakes.
     • import student grades and various information into a CSV file. The grades can then be imported into the CMS Moodle gradebook.
     • generate analysis files to create your own assessments.
